# ADR – Architecture Decision Records

Important architecture decisions are documented here.

## Index
- `0001-architecture.md` – Base architecture / layers / DB source of truth

## Creating a new ADR
1. Choose the next number (e.g. 0002)
2. Create the file: `docs/adr/0002-title.md`
3. Use this template:
   - Context
   - Decision
   - Consequences
   - Alternatives (brief)
